# Stopwatch

A simple stopwatch application created as a homework assignment for Prog Academy.

## Live Demo
https://olhakhodakivska.github.io/stopwatch/

## Description

This project implements a basic stopwatch with Start, Stop, and Reset functionality.  
Time is displayed in HH:MM:SS format and updates every second.  
The interface visually indicates the current stopwatch state using background colors.

## Technologies Used

- HTML5  
- CSS3 (Flexbox)  
- Vanilla JavaScript (ES6)

## Features

- Start / Stop / Reset timer  
- Time tracking with setInterval  
- Dynamic DOM manipulation  
- Visual state indication:
  - Green — running
  - Red — stopped
  - Silver — reset
